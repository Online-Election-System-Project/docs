# UC - 투표 결과 확인


#### 1. Related Requirements: 
REQ-16, REQ-17


#### 2. Actor's Goal: 
특정 선거의 투표 현황과 투표 결과를 확인할 수 있다.

#### 3. Initiating actor: 
일반 유저(투표자)

#### 4. Flow of Events for Main Success Scenario:

-> 1. 유저가 선거 리스트에 있는 투표 결과 확인 버튼을 누른다.

<- 2. System이 선거가 끝나기 전에는 투표 현황을, 끝나고 나서는 투표 결과와 투표 현황을 보여준다.
